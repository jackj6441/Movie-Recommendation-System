## Anti-Patterns Verdict

**LLM assessment**: Not broadly AI-looking, but two specific tells: (1) movie card grid overlay (decorative background-image grid lines) and (2) hardcoded identical signal chips on every card masquerading as dynamic data.

**Deterministic scan**: Clean — [] — zero automated violations.

## Priority Issues

**[P1] Raw API metadata exposed as user-facing copy**: anchor_source, model_version, alpha shown in Featured/Explain subtitles. Remove or move to collapsed footer.

**[P1] Cormorant Garamond used for all UI text**: Display serif at 0.8–0.85rem creates readability issues. Reserve for headings only; use system-ui for labels, buttons, inputs.

**[P1] No loading state for AI Explanation card**: RAG takes 2–6s; card shows static placeholder "when available" with no skeleton or progress indicator.

**[P2] Hardcoded identical signal chips on every movie card**: "Seed match", "Content signal", "Hybrid score" never change. Either wire to real data or remove.

**[P2] Results page has no visual hierarchy across five cards**: All cards same visual weight. "Featured for you" (primary) competes with "Explain" (debugging). Collapse score breakdown by default.

## Persona Red Flags

**Jordan (First-Timer)**: No confirmation feedback when adding a seed. Raw API strings in subtitles are uninterpretable. "AI explanation will appear here when available" reads as broken. D3 chart has no legend explaining green vs. orange bars.

**Sam (Accessibility)**: Search suggestions not keyboard-navigable (no listbox role, no arrow-key handler). Genre chips missing aria-pressed. D3 SVG has no aria-label.
